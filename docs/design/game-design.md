# 游戏设计概览

## 核心概念

太阳农场是一款轻量、明快的像素风农场养成游戏。玩家每天在农场中照料鸡舍、喂鸡、收集鸡蛋、买卖鸡和饲料，通过稳定的小目标获得成长感。

## 玩家体验目标

- 轻松：操作简单，反馈清楚，不惩罚玩家试错。
- 有节奏：每一天都有明确选择，短期行动能推动长期成长。
- 可扩展：基础农场系统能自然接入天气、建造、订单、NPC、种植和探索。
- 温暖：视觉和交互强调晴朗、活力和照料感。

## 核心循环

```text
移动到鸡舍 -> 喂鸡 -> 睡觉进入下一天 -> 获得鸡蛋 -> 去商店出售 -> 购买饲料/鸡 -> 扩大鸡舍收益
```

## 垂直切片玩法

### 移动与交互

- 玩家使用移动端触摸摇杆移动。
- 靠近鸡舍或商店后使用互动按钮。
- 键盘方向键和 WASD 保留为桌面调试输入。

### 养鸡系统

首版状态：

- 金币
- 饲料
- 鸡
- 鸡蛋
- 当天已喂鸡数量

基础规则：

- 初始金币 30、鸡 1、饲料 3、鸡蛋 0、第 1 天。
- 每只鸡每天最多喂 1 次，每次消耗 1 份饲料。
- 当天喂过的鸡会在进入下一天时各产 1 个鸡蛋。
- 鸡不会死亡、不生病、不成长，首版只做稳定闭环。

### 商店系统

首版交易：

- 买鸡：60 金币/只。
- 卖鸡：40 金币/只。
- 买饲料：3 金币/份。
- 卖鸡蛋：8 金币/个。

### 资源系统

首版资源：

- 金币
- 饲料
- 鸡蛋
- 鸡

## 初始数值

| 项目 | 数值 | 备注 |
| --- | ---: | --- |
| 初始金币 | 30 | 足够购买 10 份饲料 |
| 初始鸡 | 1 | 立即进入养鸡循环 |
| 初始饲料 | 3 | 支撑前三天产蛋 |
| 买鸡价格 | 60 | 中期扩张目标 |
| 卖鸡价格 | 40 | 可回收资源 |
| 买饲料价格 | 3 | 日常消耗 |
| 卖鸡蛋价格 | 8 | 主要收入来源 |

## 每日节奏

- 首版不做真实时钟推进。
- 玩家点击“下一天”按钮进行日结。
- 日结时根据当天已喂鸡数量产出鸡蛋。
- 后续可扩展为完整昼夜时间和体力系统。

## 画面与氛围

- 主视觉：晴朗农场、可读性高的像素角色、鸡舍和商店。
- UI：信息密度适中，优先清楚显示资源、天数、喂养状态和交互按钮。
- 色彩：使用绿色农田、金色阳光、蓝色商店和暖色鸡舍形成层次。

## 后续扩展方向

- 天气：晴天、雨天、阴天。
- 建造：仓库、鸡舍、温室、装饰物。
- NPC：商店、委托、节日。
- 制作：料理、加工品、工具升级。
- 探索：森林采集、矿洞、钓鱼。
- 种植：翻土、播种、浇水、作物成长。
