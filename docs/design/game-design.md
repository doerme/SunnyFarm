# 游戏设计概览

## 核心概念

太阳农场是一款轻量、明快的像素风农场养成游戏。玩家每天在农场中规划有限体力，完成种植、浇水、收获、整理和建设等活动，通过稳定的小目标获得成长感。

## 玩家体验目标

- 轻松：操作简单，反馈清楚，不惩罚玩家试错。
- 有节奏：每一天都有明确选择，短期行动能推动长期成长。
- 可扩展：基础农场系统能自然接入天气、建造、订单、NPC 和探索。
- 温暖：视觉和交互强调晴朗、活力和照料感。

## 核心循环

```text
选择工具 -> 处理地块 -> 播种/浇水 -> 时间推进 -> 作物成长 -> 收获 -> 出售/制作 -> 购买升级 -> 扩大农场
```

## 垂直切片玩法

### 移动与交互

- 玩家使用键盘或虚拟摇杆移动。
- 面向相邻格子执行工具操作。
- 当前工具决定交互结果。

### 农田系统

地块状态：

- 普通土地
- 已翻土
- 已播种
- 已浇水
- 成长中
- 可收获

基础规则：

- 播种需要消耗种子。
- 每天浇水后作物推进成长。
- 可收获作物进入背包。
- 长期未浇水的作物暂停成长，首版不枯萎。

### 工具系统

首版工具：

- 锄头：翻土。
- 水壶：浇水。
- 种子袋：播种。
- 手：收获和拾取。

### 资源系统

首版资源：

- 金币
- 种子
- 作物
- 体力

## 每日节奏

- 每天从 06:00 开始。
- 每 10 秒推进 10 分钟游戏时间，具体速度可配置。
- 22:00 后提示休息。
- 体力耗尽时无法使用消耗体力的工具。

## 画面与氛围

- 主视觉：晴朗农场、鲜明地块、可读性高的像素角色。
- UI：信息密度适中，优先清楚显示工具、资源和时间。
- 色彩：避免单一色调，使用绿色农田、金色阳光、蓝色水体和暖色建筑形成层次。

## 后续扩展方向

- 天气：晴天、雨天、阴天。
- 建造：仓库、鸡舍、温室、装饰物。
- NPC：商店、委托、节日。
- 制作：料理、加工品、工具升级。
- 探索：森林采集、矿洞、钓鱼。
